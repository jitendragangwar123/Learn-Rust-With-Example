#### Create First Project
```shell
$ mkdir ~/RustWithExamples
$ cd ~/RustWithExamples
$ touch hello_world.rs
```
```shell
$ rustc hello_world.rs // to compile the file
$ ./hello_world // to run the file
Hello, world!  // output
```
